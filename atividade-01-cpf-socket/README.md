Atividade 01 - Validação de CPF com Socket

Projeto desenvolvido para a disciplina de Redes utilizando Java.

Objetivo

Implementar uma aplicação cliente-servidor que valida um CPF informado pelo usuário utilizando comunicação via sockets.

Funcionamento
  - O cliente solicita um CPF via terminal
  - O CPF é enviado ao servidor
  - O servidor realiza a validação com base no algoritmo oficial
  - O resultado (válido ou inválido) é retornado ao cliente

Conceitos aplicados:
  - Programação em Java
  - Comunicação cliente-servidor (Sockets)
  - Streams (DataInputStream e DataOutputStream)
  - Estruturas de repetição e decisão
  - Manipulação de Strings
  - Validação de CPF

Como executar:
  - Execute o Servidor.java
  - Execute o Cliente.java
  - Digite um CPF quando solicitado

Observações:
  - Comunicação via localhost (127.0.0.1)
  - Porta utilizada: 50000
  - Inclui validação para CPFs com todos os dígitos iguais
