# java-redes
Projetos da disciplina de Redes em Java (UTFPR) — evolução prática com sockets e aplicações cliente-servidor.

Atividade 01 - Validação de CPF com Socket

Projeto desenvolvido para a disciplina de Redes.

📌 Objetivo

Implementar uma aplicação cliente-servidor em Java utilizando sockets para validar um CPF informado pelo usuário.

⚙️ Funcionamento
O cliente solicita um CPF via console
Envia o CPF para o servidor
O servidor realiza a validação com base no algoritmo oficial
Retorna se o CPF é válido ou inválido
🧠 Conceitos aplicados
Comunicação cliente-servidor (Sockets)
Streams (DataInputStream / DataOutputStream)
Manipulação de Strings
Estruturas de repetição e decisão
Validação de CPF
▶️ Como executar
Execute o Servidor.java
Execute o Cliente.java
Digite um CPF no cliente
📌 Observações
Utiliza localhost (127.0.0.1)
Porta: 50000
Inclui validação para CPFs com todos os dígitos iguais
